How this works. This page reads the *.jsonl transcript files you hand it with your browser's File API, tallies the swears with the same audited detector the swear-jar CLI ships, and renders this report — all in this tab. No account, no server, no upload, no cookies, no analytics. View source and check: the modules under src/ are served verbatim from the same repo you can read on GitHub.
What "sharing" sends. Nothing, until you copy the caption yourself — and the caption carries only the numbers on this page (your balance, counts, censored top word), never a single line of what you actually typed.
